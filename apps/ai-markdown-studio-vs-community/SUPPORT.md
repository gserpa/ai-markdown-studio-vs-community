# Support

AI Markdown Studio Community is provided without a support commitment. Bug reports may be reviewed at the author's discretion, but response times, fixes, compatibility, and continued availability are not guaranteed.

## Reporting an issue

If you encounter a bug, open an issue in the project repository (`https://github.com/gserpa/ai-markdown-studio-vs-community`) with:

- The Community extension version (Extensions view → ⋯ → Copy Extension Info), and VS Code version.
- Whether AI Markdown Studio Pro is also installed, and its version if so.
- Steps to reproduce, including the Markdown content or a minimal example that triggers the issue (redact sensitive content).
- Any error messages shown in VS Code notifications or the **Output** panel (select the relevant extension channel).

## Before reporting

- Check [docs/user-guide.md](docs/user-guide.md) for usage guidance on the Markdown preview, presentation preview, table formatting, themes, and HTML export.
- Check [docs/security-review.md](docs/security-review.md) if the question relates to data handling, remote resources, or content sanitization.
- If the issue only reproduces when AI Markdown Studio Pro is also installed, also check Pro's own support channel, since Pro registers additional commands and contributions on top of Community.

## Security issues

Do not file public issues for suspected security vulnerabilities. Report them privately to the author through the contact details on the Marketplace listing or repository profile.
