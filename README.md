# AI Markdown Studio Community

AI Markdown Studio Community is the MIT-licensed open-source core of AI Markdown Studio.

It provides preview-first Markdown authoring, live document and presentation previews, Mermaid, KaTeX, syntax highlighting, table formatting, themes, speaker-note previews, AI-assisted document and presentation generation, AI Paste to Markdown, and HTML and basic DOCX export.

AI theme workflows, PDF/PPTX export, high-fidelity DOCX, file conversion, agent tools, and corporate-template automation are provided separately by AI Markdown Studio Pro. Pro depends on Community; Community contains no Pro implementation.

## Development

```powershell
npm install
npm run verify
```

The Community boundary check rejects Pro commands, dependencies, source paths, and packaged files.
